# Sistema de Video Wall - SALA NOC

Sistema de gerenciamento de video wall para 56 monitores.

## Como baixar o executável

1. Vá na aba **Actions**
2. Clique no workflow mais recente
3. Baixe o arquivo **VideoWall_NOC_Windows**
4. Extraia e execute o .exe

Não requer instalação nem privilégios de administrador!
